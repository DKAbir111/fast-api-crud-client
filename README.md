# 🚀 FastAPI CRUD Client (Learning Project)

This is a simple **FastAPI CRUD API project** created for learning backend development with Python and FastAPI.

👉 Repository: https://github.com/DKAbir111/fast-api-crud-client

---

## 📌 Purpose

This project is built for practice to understand:

- FastAPI framework basics
- REST API structure
- CRUD operations
- Request & response handling
- API documentation using Swagger UI

---

## 🧠 What I Learned

While building this project, I practiced:

- ⚡ FastAPI setup and routing
- 📦 CRUD operations (Create, Read, Update, Delete)
- 🧾 Pydantic models for validation
- 🔗 RESTful API design
- 🧪 Interactive API testing with Swagger (`/docs`)
- 🏗️ Backend project structure basics

---

## 🛠️ Tech Stack

- Python 🐍
- FastAPI ⚡
- Uvicorn
- Pydantic

---

## 📂 Features

- Create new items
- Get all items
- Get item by ID
- Update items
- Delete items
- Auto-generated API documentation
